## Sobre

Projeto pessoal de apresentação: uma página interativa que conta quem é o **SrChZ** através da música — playlists, gêneros e referências que formam a trilha sonora pessoal, com toques de gastronomia e identidade.

> *"Não é sobre o que eu faço — é sobre quem eu sou."*

Slowed + reverb, phonk, trap BR. 2.934 músicas curtidas destiladas em uma experiência única.

## Tecnologia

Página estática (`HTML` · `CSS` · `JavaScript`), sem dependências de build, publicada no **GitHub Pages**.

## Rodar localmente

```bash
python -m http.server 8000
# abra http://localhost:8000
```
